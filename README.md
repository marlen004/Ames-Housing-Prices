THIS PROJECT IS PART OF A PORTFOLIO, ITS ENTIRE PURPOSE IS TO SHOW OFF MY CODING SKILLS AND UNDERSTANDING OF DATA PIPELINES AND DATA ANALYSIS
THE MAIN ETL SCRIPTS ARE INSIDE THE SCRIPTS FOLDER, TOGETHER WITH THE LOGGING CONFIGURATION
MAIN TOOLS FOR ANALYSIS ARE INSIDE THE ANALYSIS FOLDER, A SHORT EDA JUPYTER NOTEBOOM AND ANALYTICS.PY FILES
DATA IS FROM THE AMES HOUSING DATASET, RAW AND CLEANED VERSIONS ARE STORED INSIDE THE DATA/CLEAN AND DATA/RAW FOLDERS
THE RESULTS AS MODELS OR MAJOR DECISIONS/TEST RESULTS ARE STORED INSIDE THE RESULTS FOLDER
PLOTS AND DASHBOARDS ARE INSIDE THE VISUALIZATION FOLDER
IT'S RUN ENTIRELY ON A MYENV VIRTUAL ENVIRONMENT, ALL NECESSARY MODULES AND VERSIONS ARE STORED IN REQUIREMENTS.TXT FILE
TO RUN THE PROJECT, EXECUTE PYTHON3 MAIN.PY AFTER ACTIVATING THE VENV OR CONDA OR WHATEVER ENVIRONMENT YOU USE. DO NOT FORGET, IMPORTANT DECISIONS ON COLUMNS WERE MADE IN THE EDA ANALYSIS, SO READ IT BEFORE RUNNING EVERYTHING
